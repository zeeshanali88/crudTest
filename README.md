## How to setup
Copy .env.example and rename it to .env and update the DB config variables.
Run the following commands to run the project locally:
```angular2html
composer install //To install dependencies
php artisan migrate //Create Db structure 
php artisan db:seed //Add some default data to db (i.e brands)
php artisan serve
```
You can view the site by going to http://127.0.0.1:8000
